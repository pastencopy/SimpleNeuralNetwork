# SimpleNeuralNetwork
Simple NeuralNetwork for Study

+NerualNetworkLibrary : 신경망네트워크와 행렬 라이브러리

+SimpleNeuralNetwork : 신경망네트워크를 이용한 예제 , Xor연산을 지도학습으로 예측

+SimpleRocket : 유전알고리즘을 이용한 목표물에 이동하는 로켓


공부하면서 만드는 간단한 신경망 네트워크 알고리즘입니다.
