# SimpleNeuralNetwork

2020 Coded by GwangSu Lee

Visual Studio 2019 Community 버전에서 C#으로 구현되어 있습니다.



NerualNetworkLibrary : 신경망네트워크와 행렬 라이브러리

SimpleNeuralNetwork : 신경망네트워크를 이용한 예제
  + Xor연산을 지도학습으로 예측

SimpleRocket : 유전알고리즘을 이용한 목표물에 이동하는 로켓
